Welcome to ASCII and EBCDIC and more information about them.

To represent numeric, alphabetic and special character in a computer internal storage and magnetic media, we must use some sort of coding system. in computer, the code is made up of a fixed size group of binary. The ASCII and EBCDIC both use the 8 bit, yet ASCII also use 7 bit before it was extended.

ASCII

ASCII The acronym od ASCII American Standard Code for information Interchange in ASCII the letters are broken into two groups, unlike EBCDIC. Upper cases are assigned codes beginning with hexadecimal values 41 and continuing sequentially through hexadecimal value of 5A Similarly, lowercases latters are assigned hexadecimal values of 61 through 7A. The decimal 1 through 9 are assigned the zone code 0011 in ASCII rather than 1111 as in ASCII
