A character is any latter, symbol, number, space, punctuation or mark that can be typed in the computer. The word "computer" for example consist of 8 characters. The phrase "Hi there" takes up nine characters.

For example the word "hamburger" and the phrase I ate three humburgers are both string even "12345" can be considered as a string if specified correctly.

BOOLEAN

A boolean, or boolean logic is a subset of algebra used for creating True or False statement. Boolean expresson use the operation AND, OR, AND, NOT to compare values and return to the true of false results.These boolean operaters are discribes in the following four examples.

  1. x AND y - return true if both x and y are true and return false if both x and y are false
  2. x OR y- return true if x and y or both x and y are true, return false if x and y are both false
  3. x XOR y- return true if only x and y is true, return false if x and y are both true or both false
  4. NOT x - return true if x is false ("or null"), return false if x is true

Since computers operates in binary (using 1s and 0s, computer logic can often express in boolean terms. For example, a true statement can be represented by 1 and false statement can be represented by 0
